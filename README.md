# FizzBuzzTraining
http://qiita.com/tadsan/items/b9521f149d4964b506d0


# requirement

* PHP 7.0 or later
* composer

# Installing

```bash
git clone git@github.com:ndxbn/FizzBuzzTraining.git
cd FizzBuzzTraining
composer install
```

# Testing

```bash
composer test
```

# Build-in Web Server

`composer serve` and access to http://localhost:8080
